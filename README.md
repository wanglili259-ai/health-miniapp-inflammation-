# 健康炎症风险评估小程序

一款基于微信小程序的实用工具，帮助用户快速收集健康信息并评估身体炎症风险等级，获取个性化健康建议。

## 功能简介

| 页面 | 功能 |
|------|------|
| 首页 (`index`) | 应用介绍、评估内容概览、入口按钮 |
| 问卷页 (`questionnaire`) | 13 道问题的健康信息采集（支持进度条） |
| 结果页 (`result`) | 评分展示、风险等级、个性化建议 |

## 评估维度

- **基本信息**：年龄段、体重（BMI）
- **身体症状**：关节疼痛、疲劳感、消化不适、皮肤问题
- **饮食习惯**：加工食品、糖分摄入、蔬果摄入
- **生活方式**：运动频率、睡眠质量、压力水平、吸烟情况

## 评分算法

每题 0–3 分，满分 39 分。

| 风险等级 | 分数区间 |
|----------|----------|
| 🟢 低风险 | 0 – 12 分（≤ 33%） |
| 🟡 中等风险 | 13 – 25 分（34–67%） |
| 🔴 高风险 | 26 – 39 分（≥ 68%） |

## 目录结构

```
├── app.js                          # 全局逻辑（数据存储）
├── app.json                        # 全局配置（页面注册）
├── app.wxss                        # 全局样式
├── sitemap.json
├── utils/
│   └── evaluate.js                 # 评分工具函数（核心逻辑）
└── pages/
    ├── index/                      # 首页
    ├── questionnaire/              # 健康问卷页
    └── result/                     # 评估结果页
```

## 使用方法

1. 使用微信开发者工具打开项目根目录
2. 确保 AppID 配置正确（可使用测试号）
3. 点击编译预览即可运行
